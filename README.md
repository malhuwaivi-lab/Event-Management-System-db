## Overview
A robust database system designed to handle complex event scheduling, user transactions, and attendee management. This project focuses on database normalization and data integrity to ensure efficient retrieval for large-scale event data.

## Key Features
* **Normalized Schema (3NF):** Designed a relational database in Third Normal Form to eliminate redundancy and prevent data anomalies.
* **Analytics Dashboard:** Implemented optimized SQL queries to track engagement metrics and attendee statistics.
* **Data Integrity:** Enforced strict referential integrity constraints to maintain accurate relationships between Users, Events, and Bookings.
* **Complex Queries:** Utilized Joins and Aggregations to generate real-time reports.

## Tech Stack
* **Language:** Java, SQL
* **Database:** Relational Database Management System (RDBMS)
* **Concepts:** 3NF Normalization, ER Modeling, Query Optimization
